## AI/ML Roadmap

1. **Learn Python and Core Libraries**
    - [Intro Python - Harvard CS50](https://cs50.harvard.edu/python/2022/)
    - [Advanced Python - Harvard AI with Python](https://www.edx.org/learn/artificial-intelligence/harvard-university-cs50-s-introduction-to-artificial-intelligence-with-python)
    - [NumPy Quickstart](https://numpy.org/devdocs/user/quickstart.html)
    - [Pandas Tutorial](https://www.w3schools.com/python/pandas/default.asp)
    - [Matplotlib Tutorials](https://matplotlib.org/stable/tutorials/index.html)
    - [Scikit-learn Tutorial](https://scikit-learn.org/1.4/tutorial/index.html)

2. **Build a Strong Math Foundation**
    - [Linear Algebra - MIT](https://ocw.mit.edu/courses/mathematics/18-06-linear-algebra-spring-2010/)
    - [Probability & Statistics - Stanford](https://web.stanford.edu/class/stats116/syllabus.html)
    - [Multivariable Calculus - Khan Academy](https://www.khanacademy.org/math/multivariable-calculus)

3. **Learn ML Fundamentals**
    - [Google ML Crash Course](https://developers.google.com/machine-learning/crash-course)
    - [Machine Learning by Andrew Ng](https://www.coursera.org/learn/machine-learning)
    - [The Hundred-Page ML Book](http://ema.cri-info.cm/wp-content/uploads/2019/07/2019BurkovTheHundred-pageMachineLearning.pdf)

4. **Build Practical Experience**
    - [Practical Deep Learning for Coders - fast.ai](https://course.fast.ai/)
    - [Structured ML Projects - Coursera](https://www.coursera.org/learn/machine-learning-projects)
    - [Build GPT from Scratch - Karpathy](https://www.youtube.com/watch?v=kCc8FmEb1nY)

5. **Specialize**
    - [NLP Course - Hugging Face](https://huggingface.co/learn/nlp-course/chapter1/1)
    - [Deep RL Course - Hugging Face](https://huggingface.co/learn/deep-rl-course/unit0/introduction)
    - [Computer Vision - Kaggle](https://www.kaggle.com/learn/computer-vision)
    - [Deep Learning - CS231n Stanford](https://www.youtube.com/watch?v=vT1JzLTH4G4&list=PLSVEhWrZWDHQTBmWZufjxpw3s8sveJtnJ)

6. **Learn MLOps**
    - [Intro to MLOps](https://ml-ops.org/)
    - [Three Levels of ML Software](https://ml-ops.org/content/three-levels-of-ml-software)
    - [Full Stack Deep Learning](https://fullstackdeeplearning.com/course/2022/)

7. **Read Research Papers**
    - [ArXiv](https://arxiv.org/)
